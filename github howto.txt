type 'git init' (no quotes) in an empty folder to setup the repo. then use any of
the following to add

git remote add origin https://github.com/turtle-master/TurtleGithub.git


FIRST change directory to your repo (open gitbash)
add whatever files to the folder. then type in git bash:

git add .
git commit -m 'some comment here'
git push origin master


to remove:
git rm file.txt
git commit -m 'asdf'

updating to folder:
delete any files from the folder that you dont want. then:
git add . -A
git commit -m 'some comment here'
git push origin master
